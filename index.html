Dual-Path Convolutional Image-Text Embeddings with Instance Loss

Posted on 2018-11-21

摘要部分：
这篇论文提出了一种新的系统，可以将图像和文本有区别地嵌入到共享的视觉文本空间中。在人物搜索的方法中，大部分现在方法都是使用排名损失（ranking loss）来使得正确的图像-文本对更加接近，而错误的图像-文本对更加远离的方法。然而，这些方法有一个问题就是直接部署排名损失对于网络学习比较困难，因为它从两个不同结构的特征开始构建模态间的关系。为了解决这个问题，这篇论文提出了明确地考虑到模态内数据分布的实例损失函数（instance loss）。这是基于一个无监督的假设，也就是说，每一个图像-文本对能够被看作一个类别，因此网络可以从每个图像-文本对中学习到细粒度。这个方法的优点被证明可以提供一个更好的权重初始化，能够学习更多的判别嵌入。

介绍：
图像和文本都包含非常丰富的语义信息，但是处于不同的模态。和相同模态内的信息检索相比，图像和文本的匹配有着额外的关键挑战，也就是说图像和文本映射到一个共享的特征空间，所以这个任务就需要通过很强的特征表示来连接不同的表示。
在过去的几年中，ranking loss常被用作图像-文本表示学习的目标函数，这个损失函数旨在使得正样本对的距离尽可能的小，而负样本对的距离尽可能的大。在图像和文本的匹配任务重，每个训练对包含视觉特征和文本特征，ranking loss侧重于两种方式之间的距离，它的潜在缺点是它没有明确的考虑单一模态中的特征分布。在人物搜索的任务中，还有一个难点就是极细粒度，要求对于描述的语句能够精准的找到top-1的人。这篇论文发现了单独使用ranking loss可能会导致网络陷入局部极小值。

Person Search with Natural Language Description

Posted on 2018-11-21

本文方法：
GNA-RNN将描述语句和人物图像作为输入并输出它们之间的亲和度。将句子输入到单词LSTM中并逐字处理。在每个单词中，LSTM为单个视觉单元生成单元级注意，每个视觉单元确定输入图像中是否存在某些人的语义属性或视觉模式。视觉单元注意机制对不同单词的不同单元的贡献进行加权。此外，我们还学习了词级门，用于估计不同词对于自适应词级加权的重要性。通过对所有单位的所有单词的响应进行平均来获得最终的亲和力。单元级注意和字级S形门都有助于我们提出的GNA-RNN的良好性能。
本文工作和贡献：

  1. 建立了一个大规模的语言描述搜索行人的数据集，有13003个人的40206张图片；
    这个数据集具有以下特点：一是人员都是采用的不同场景下的行人，行人图像的内容很多样；二是在语言描述方面，共有80412条句子，丰富的词汇短语句子模式结构，也是十分多样化。（合并了五个reid数据集，viper+cuhk01+cuhk03+market1501+ssm，注释文件中包括了两个自然语言描述，行人的id1-13003，文件路径）
    【 ssm: End-to-end deep learning for person search】
  2. 提出了一种门控神经注意的循环神经网络，用来描述语句和任务图像作为输入并输出他们之间的亲和度。
  3. 基于不同的视觉和语言框架研究了各种合理的解决方法，image caption，视觉语义嵌入等，并给行人搜索问题建立了基准。

Cascade Attention Network for Person Search Both Image and Text-Image Similarity Selection

Posted on 2018-11-21

摘要部分：
使用自然语言的人物搜索，是通过关于人的描述句子来检索图像数据库中的对应的人物，这对于许多应用很有价值，比如在视频监控领域就可以发生巨大的作用。这其实是一个跨模态的匹配问题，而如何将视觉内容提取到人类描述是这个问题的关键部分。
这篇论文里面，提出了一个级联注意力网络（Cascade Attention Network，CAN）来逐步从人物图像和文本图像相似性中进行选择。
在CAN网络中，首先提出了姿势引导注意力来关注增强输入中的人物，其将原始3个图像通道和另外14个姿势的置信图链接起来；然后通过提取得到的人物图像表示，计算了人物部分和文本描述之间的局部相似性，然后提出基于相似性的强烈关注以进一步从那些局部相似性中选择与描述相关的相似性得分。
实验部分是在CUHK-PEDES数据集上做的，这个是目前唯一使用自然语言进行人员搜索的数据集。

介绍：
人物搜索近年来因其在视频监控中的广泛应用而受到极大的关注，例如，丢失人物搜索和嫌疑犯追踪。随着监控视频的数量大幅度增加，在大型数据集手动搜索人物是不现实的，因此，我们需要设计方法来有效的执行任务。
目前的任务搜索方法主要根据查询的类型分成三类，第一类是基于图像的查询，第二类是基于属性的查询，第三类是基于文本的查询。基于图像的人物搜索至少需要一个被查询人的图像，在许多情况下难以获得。基于文本的人物搜索可以解决人物图像遗漏的问题，因为文本描述更易于访问。此外，与基于属性的人物搜索相比，基于文本的方法可以以更自然的方式描述具有更多细节的人。因此研究用自然语言进行人物搜索的任务是一件有意义的事情。
用自然语言描述搜索人物的主要挑战之处在于基于人类的描述提取出相对应的视觉内容，并且在特征空间中构建它们的映射。
【图像特征表示】以前的方法是使用卷积神经网络获取输入图像的全局表示，这通常无法有效提取与图像中人物相对应的视觉内容。考虑到人体姿势和人体部位密切相关，这篇论文利用姿势信息来引导人们视觉特征选择的注意，同时这也是目前来说首次采用人体姿势来处理基于文本的人物搜索的工作。
【图像和文本匹配策略】以前的工作提出了一些利用局部特征进行精确匹配的方法，还有的方法融合了所有文字对应图片的相似之处，另外还有工作是计算短语和图像的相似度来加权图像特征图。这篇论文是计算人物图像区域和文本之间的局部相似性，为了进一步选择和文本描述中最相关的人物部分，我们采用了强关注对于局部的相似性。
【本文方法介绍】在这篇论文中，我们提出了一种用自然语言进行人物搜索的级联注意网络（CAN），它可以从人物图像和文本-图像相似性中逐步选择关键匹配线索。
首先，作者从输入图像估计人体姿势并提取对应14个身体关键点的14个置信图。一方面，置信图和输入图像相结合来增强视觉表示，另一方面，置信图用于引导人物特征选择的注意力。
具体来说，作者在这篇论文中采用了视觉CNN，比如VGG-16和ResNet50和pose-CNN来分别提取视觉表现和姿势表示。然后结合这两种表示，可以获得对视觉表示施加权重的姿势注意力图，这个加权视觉表示强调了人在图像中的重要性。然后使用了BiLSTM网络来学习文本描述的表示，考虑到文本描述通常很长，并且包含几个短语，作者采用所有短语的综合表示来表示整个描述。
最后，作者计算人物图像区域与文本描述之间的局部相似性，而不是图像和文本描述之间的全局相似性，为了进一步选择与文本描述最相关的人类部分，作者定义了一个阈值，来对局部的相似性进行硬关注。在网络中的识别损失和排名损失对于训练会有很好的作用。
【贡献】这篇论文的贡献主要有三点：

  1. 作者建议利用姿势信息来指导人物视觉特征选择的注意，这个方法首次用在基于文本的人物搜索，实验效果也不多。
  2. 作者提出了基于相似性的强关注（hard attention），来进一步选择和文本描述最相关的人物部位。
  3. 作者提出了级联注意网络在具有挑战性的数据集CUHK-PEDES上实现了最佳的效果，并且验证了CAN每个组件的有效性。

相关工作：
这篇文章的相关工作主要包括：使用自然语言描述进行人物搜索，利用人物姿势进行人物搜索，使用注意力机制进行人物搜索。
【用自然语言搜索人物】li2017b提出了用自然语言进行人物搜索的任务，并进一步提出了一种具有门控神经注意力的循环神经网络（GNA-RNN）来完成这项任务。为了利用身份级别的注释，li2017a提出了一个基于身份识别的两阶段框架，两个阶段都是使用的CNN-LSTM网络框架，第一阶段实现嵌入跨模式特征，第二阶段利用潜在的共同注意力机制来细化网络模型。Chen2018是提出了一种文字和图像patch匹配的模型，来捕获局部相似性。Zheng2017b提出了采用CNN进行文本特征学习，也就是textCNN。

【利用人物姿势进行人物搜索】基于图像的人物搜索方法中，有一些方法利用人类姿势来提高图像的视觉表现，主要用到的方法是利用姿势来规范人形象，通过标准化图像和原始图像来匹配人物。

【注意力机制来进行人物搜索】注意力机制只要是指选择输入的关键部分，通常分为软注意和硬注意（重视），软注意只是计算权重图并根据权重图来选择输入，而硬注意只保留一个或者几个部分的输入，而忽视其他部分。注意力机制用在人物搜索上，主要是选择关键的视觉内容和文本信息。

person re-identification 论文阅读记录

Posted on 2018-11-07

1.Video-based person re-identification via self-paced weighting论文阅读记录
总体是分为三步：
第一步，序列鲁棒性度量（sequence stability measure，ssm）信号用来检测视频序列的状态变化并且自动将未经调节的视频序列分解为多个片段。
第二步，使用spod方法来评估每一个子序列的噪声度。
第三步，使用一个加权的多对距离度量学习方法被提出来联合度量两个人图像序列的距离。

核心idea：将图像序列中的噪声作为参数加入到图像序列距离匹配的计算当中，当噪声参数比较大的话，这个噪声在测量视频序列距离时，将在某种程度上被消除。

2.Video Person Re-identification with Competitive Snippet-similarity Aggregation and Co-attentive Snippet Embedding
核心idea：竞争性片段相似性聚合和共同注意片段嵌入

3.Multi-shot Pedestrian Re-identification via Sequential Decision Making
核心idea：强化学习来做视频序列化的决策，给定一个probe行人图像，通过比较gallery中一个视频序列中不同的样本和这个prebe 的特征差别，然后给出三个判断，两段视频是否相同或者无法判断，当视频序列帧中图像无法判断时，则推移到下一个视频帧进行比较。

4.Mask-guided Contrastive Attention Model for Person Re-Identification
核心idea：提供一个二值化标记，将背景部分去除，得到一个仅剩行人的图像。通过结合只有行人，行人加背景的图像以及只有背景的图像，使用一个掩模引导的对比注意模型来进行行人再识别。主要的优点在于使得背景干扰尽可能的小。

5.Diversity Regularized Spatiotemporal Attention for Video-based Person Re-identification
核心idea：主要考虑到现实世界中行人再识别遮挡的情况极容易出现，所以提出了一个时空注意力模型来自动发现视频中行人独特的身体部分（脸、背包等），然后网络通过学习多个空间注意力模型以及使用多样化的正则项来保证多个模型不会发现同样的身体部位，最后提取出来的局部特征使用暂时注意力模型结合指导行人再识别。

6.Jointly Attentive Spatial-Temporal Pooling Networks for Video-based Person Re-Identification
核心idea：提出了一个联合时空注意力池化网络，能够使得特征提取器关注当前输入的视频序列，在某种程度上，匹配的关注点的相互依赖性可以直接影响彼此表示的计算。另外，空间池化层能够在每一帧中选择区域，同时执行时间注意力池化可以在视频序列里面选择信息帧，两者都是由距离匹配的信息做引导的。
